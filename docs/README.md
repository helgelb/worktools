# Documentation

This directory contains detailed documentation for all tools in this repository, organized by language.

## Structure

```
docs/
└── python/               # Python tools documentation
    └── allocate_hours.md # Hours allocation tool guide
```

## Documentation Guidelines

- Keep documentation simple and practical
- Focus on usage examples rather than implementation details
- Include both CLI and programmatic usage where applicable
- Provide clear examples with expected output

## Adding New Documentation

When adding tools to the repository:

1. Create language-specific subdirectories (e.g., `python/`, `javascript/`)
2. Create detailed documentation files for each tool
3. Update the main tool README to point to the documentation
4. Follow the existing documentation style and structure
